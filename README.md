## 🛠️ Tech Stack

- **AI Brain**: Groq API (Llama 3) — FREE
- **Backend**: Python · FastAPI · APScheduler
- **Frontend**: React · Vite · TailwindCSS
- **Integrations**: Gmail · Google Calendar · Meta Insights · Notion · Telegram
- **Deploy**: Vercel (frontend) · Railway (backend)
